How to get Wi-Fi Password ?

Yeah, finally we are into Wifi Hacking also. Today we will learn to hack WiFi using some tools, scripts, and commands. Let's understand it... Today we are going to use social engineering to hack WiFi passwords. It's similar to the phishing attack that we used to do for hacking Instagram and Facebook. Here, we are going to block the internet of the WiFi user until that user enters the Wi-Fi password to enable a fake router firmware update(our phishing attack).

Using the Wifiphisher tool
You can get more info at https://www.kali.org/tools/wifiphisher/. we can select any Wi-Fi network and de-authenticate all users (jam it), then we will create a fake access point that requires no actual or real password to join. Any user that connects to our fake network will go through a convincing phishing page demanding the Wi-Fi password to enable a firmware update, which is explained as the reason the Wi-Fi has stopped working. Are you getting the point? first, we will jam the network, and then we will ask them for the password :) Let's start it...

Step 1 :

For this attack, you'll need a Kali Linux compatible wireless network adapter with monitor mode in it. In your Kali Linux, you should first update by running apt update and apt upgrade to install our tool in the next step.

Step 2 :

We need to install Wifiphisher by entering the following command in the terminal:

apt install wifiphisher

Step 3 :

Now plug in the wireless network adapter and wifiphisher will automatically put your card into wireless monitor mode.

Step 4 :

Time to run the script...if you are using a USB wireless network adapter, you will need to add an -i flag to the command and add the name of your network adapter. To start the script, I'll run the following command:

wifiphisher -i wlan1

Step 5 :

The above command will be showing every nearby network. You can select which network you want to attack, and simply press Enter.

Step 6 :

Now, the script will ask which attack you want to perform. Select option 2 (Firmware Upgrade).

Step 7 :

After selecting the type of attack, it will launch smoothly. A page will open that will monitor the targets joining the network. Wifiphisher will also listen for devices trying to connect to networks that aren't present, and it will create fake versions to tempt those devices into connecting. Samjh rahe ho, kitna tagda game khel rahe hai ham yahan :)

After any target joins the network, they will see a pop-up demanding the password to update the firmware. When the target enters the password, you will be notified on the Wifiphisher screen.

And that's it, babes! the script will finally exit and give you the password you just captured from all the users who got trapped in the firmware update phishing popup.

This is the easiest way to hack Wifi-Passwords. Ise try kar dala to life jinghalala...
